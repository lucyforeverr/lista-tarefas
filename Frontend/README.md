// Lista de Tarefas

Projeto desenvolvido para a disciplina de Desenvolvimento Full Stack.

// Tecnologias

- React
- Node.js
- Express
- Prisma ORM
- SQLite
- JWT
- bcrypt

// Funcionalidades

- Cadastro de usuários
- Login
- Autenticação com JWT
- CRUD de tarefas
- Cada usuário acessa apenas suas próprias tarefas

// Backend

cd Backend
npm install
npm run dev

//Frontend

cd Frontend
npm install
npm run dev
